# Notachain

A protocol for legally-anchored on-chain attestations.

Prove any fact about yourself on-chain — signed by a credentialed legal professional, owned by you, and you earn every time it's read.

## What is this?

Notachain lets solicitors, notaries, barristers, and other credentialed legal professionals attest off-chain facts on-chain. Each attestation is a boolean: true or false. The proof lives on IPFS. The signature goes on-chain. Your identity is never exposed.
